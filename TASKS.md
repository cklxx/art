# 项目任务状态

本页面梳理已完成的工作和待完成的事项，便于快速了解当前进展并指向对应的技术路径。

## 已完成
- 完成 `agent` 命名统一与入口整理，LLM 客户端保持可拔插（OpenRouter/Gemini/本地）。
- 建立文本、图片、音频的标准化摄取与 `KnowledgeBundle` 规范，并接入多模态混合推理图。
- 上线总结、决策支持、论文评审等 API，内置轻量评测与追踪组件保证迭代闭环。
- 提供内存/SQLite/JSONL/文件对象存储等适配器，并通过 `/adapters`、`/traces/*` 便捷观测状态。
- 前端仪表盘呈现路标数据、适配器健康度、追踪时间线与检索沙箱，支持无密钥体验。
- 覆盖多模态小样本的检索基准集与运行器，可对比不同适配器/引擎的精度。
- 给出检索→综合→评测的多 Agent 编排示例，并通过 API 对外暴露。
- 自动化跑批检索基准，支持对多种检索适配器的精准度/召回率趋势跟踪。
- 前端补充编排回放与自定义目标入口，可一键重放 orchestration 并查看评测结果。
